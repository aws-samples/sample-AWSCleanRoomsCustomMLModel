# Synthetic Demo Data

The CSV files in this directory are **100% synthetic** — generated by `generate_synthetic_data.py` for demonstration purposes only.

## Data Provenance

| Property | Value |
|----------|-------|
| Source | Algorithmically generated using Python `random` module (seed 42) |
| Contains real PII | No |
| Contains real customer data | No |
| Contains real transaction data | No |
| Safe for public distribution | Yes |

## Files

| File | Description | Approximate Rows |
|------|-------------|-----------------|
| `advertiser_engagement.csv` | Simulated ad engagement data (impressions, clicks, time spent) | ~18,000 |
| `retailer_purchases.csv` | Simulated purchase data (amounts, counts, conversion flag) | ~22,500 |

## Regeneration

To regenerate the data with the same seed (deterministic output):

```bash
python data/generate_synthetic_data.py
```

The `user_id` values (e.g., `user_000000`) are synthetic identifiers with no connection to real individuals.
